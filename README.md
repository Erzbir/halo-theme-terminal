# Halo Terminal

一款 Terminal 风格的 Halo 主题

基于 wan92hen 的 [Terminal](https://github.com/wan92hen/theme-terminal)
修改, 使得更接近原版 [Hugo Terminal](https://github.com/panr/hugo-theme-terminal)

![demo](assets/images/demo.png)

## 特性
- 明暗模式
- 首页公告
- 评论
- 友链
- 错误页
